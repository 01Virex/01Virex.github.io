# 文章图片目录

文章的**封面图**和**正文插图**都放这儿(`static/img/posts/`)。

## 封面图
在文章顶部的信息区(`---` 之间)加一行 `cover`:

```yaml
---
title: "我的文章"
date: 2026-06-18T09:00:00+08:00
cover: "/img/posts/我的封面.jpg"
---
```
封面会显示在文章顶部(大图),首页和文章列表的卡片上显示小缩略图。

## 正文插图
在正文里用标准 Markdown 语法,方括号里的文字会变成图片下方的**说明**:

```markdown
![这是图片说明](/img/posts/某张图.jpg)
```
图片自动带说明文字、懒加载,点击可放大。

- 路径都以 `/img/posts/` 开头。
- 支持 jpg / png / webp / gif / svg。
